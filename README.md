# Election Process Education

An interactive educational web application designed to explain the election process in a simple and engaging way.

## Features
- Step-by-step Election Timeline
- FAQ Section
- Interactive Quiz
- Myth vs Fact Section
- Responsive Design

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- Docker
- Google Cloud Run

## Project Structure
election-guide/
├── index.html
├── style.css
├── script.js
├── server.js
├── package.json
└── Dockerfile

## Run Locally
1. Install Node.js
2. Run:
   npm install
   npm start

3. Open browser:
   http://localhost:8080

## Deployment
Deployed using Google Cloud Run
